Fails or allows a submission based on set conditions.

== Description ==

A form processor for [Caldera Forms](https://wordpress.org/plugins/caldera-forms/) that allows you to set up certain conditions that will make a submission fail or succeed. This can be used to disallow submissions from users with .yahoo or .live email addresses or simply disallow submissions from people named Josh.

A free plugin by <a href="https://CalderaWP.com" title="CalderaWP: Transform Your WordPress Experience">CalderaWP</a>.

== Installation ==

Install the plugin through the WordPress Plugins Installer or upload the extract folder to the plugins directory of your content directory
1. Activate the plugin through the 'Plugins' menu in WordPress
